# Indian Language News Summarization System

Abstractive News Summarization for Indian Regional Languages using Transformer Models.

---

## 📌 Project Overview

This project builds an **automatic news summarization system** that generates concise summaries of long news articles written in Indian languages.

The system uses a **Transformer-based multilingual model (mT5)** to generate **abstractive summaries** and compares the results with an **extractive summarization baseline**.

---

## 🚀 Features

- Multilingual News Summarization
- Abstractive summarization using Transformer models
- Extractive summarization baseline
- Text preprocessing pipeline
- ROUGE and BLEU evaluation metrics
- Visualization of model performance
- Word frequency analysis
- Word cloud generation

---

## 🛠 Technologies Used

- Python
- PyTorch
- HuggingFace Transformers
- NLTK
- Pandas
- Matplotlib
- WordCloud
- Gradio

---

## 🤖 Model Used

Model: **mT5 Multilingual XLSum**

This model supports summarization across multiple languages including Indian regional languages.
